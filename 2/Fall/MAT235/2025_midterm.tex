\noindent
\textbf{Q1)} Given two differential equations, we have
\[\frac{dy}{dx}+2y=z,\qquad \frac{dz}{dx}+2z=e^{-2x},\]

\noindent where $y=y(x),\: z=z(x)$.

\hfill

\noindent Find $y$ explicitly. Further, we have $\dfrac{dy}{dx}=0$ at $(x,y)=(0,1)$.

\noindent
\textbf{Q2)}
We have the initial value problem
\[(t^2+ty)y'=y^2-ty,\quad t>0,\]

\noindent with $y(1)=0$.

\hfill

(a) State the exactness, order, linearity.

\hfill

(b) Find the solution to this IVP. Explain.

\noindent
\textbf{Q3)}
We have the differential equation
\[2y(1+x^2)y'+xy^2=(1+x^2)^{3/2}, \qquad y(0)=0.\]

(a) Check the exactness.

\hfill

(b) If not exact, find an integrating factor and solve the problem.
